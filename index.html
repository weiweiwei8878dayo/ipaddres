IPアドレスとは

インターネットサービスプロバイダーは、インターネットプロトコル（IP）アドレスと呼ばれる数値ラベルを割り当てて、何十億ものデバイスの中からお手持ちのデバイスを識別します。デバイスはIPを使用して相互に検索および通信するため、IPアドレスはオンラインホームアドレスとして機能します。

ここでは、IPアドレスが宛先にデータを送る方法を説明します。まず、ブラウザにウェブサイト名（example.com）を入力します。しかし、コンピューターは単語を理解せず、数字しか理解しません。そのため、まずそのWebサイトのIPアドレス（example.com = 103.86.98.1。）を見つけ、それをWeb上で見つけ、最後に画面にロードします。

IPアドレスを見つける方法

IPアドレスを見つけるのはとても簡単です。見つけ方の例をいくつかご紹介します。

自分のパブリックIPアドレスを見つける方法

パブリックIPアドレスを見つけるには、このページの上部までスクロールします。IP、接続元の場所、サービスプロバイダー、トラフィックが保護されているかどうかが表示されます。

Windows 11で自分のIPアドレスを確認する方法

Wi-Fiに接続している場合、Windows 11で自分のIPを確認するには次の手順に従ってください。

  1. [スタート] > [設定] > [ネットワークとインターネット] > [Wi-Fi]の順にクリックします。
  2. 接続しているネットワークを選択します。
  3. [プロパティ] をクリックします。
  4. ウィンドウの一番下までスクロールします— [IPv4アドレス] の横にIPが表示されます。

イーサネットネットワークに接続している場合、IPを見つける方法：
  1. タスクバーで、[イーサネットネットワーク] アイコンをクリックします。
  2. 接続しているネットワークを選択します。
  3. [プロパティ] をクリックします。
  4. ウィンドウの一番下までスクロールします— [IPv4アドレス] の横にIPが表示されます。

Windows 10で自分のIPアドレスを見つける方法

Wi-Fiに接続している場合、Windows 10でIPを見つける方法：

  1. タスクバーの [Wi-Fiネットワーク] をクリックします。
  2. 接続しているネットワークを選択します。
  3. [プロパティ] をクリックします。
  4. ウィンドウの一番下までスクロールします— [IPv4アドレス] の横にIPが表示されます。

イーサネットネットワークに接続している場合、IPを見つける方法：
  1. タスクバーで、[イーサネットネットワーク] アイコンをクリックします。
  2. 接続しているネットワークを選択します。
  3. [プロパティ] をクリックします。
  4. ウィンドウの一番下までスクロールします— [IPv4アドレス] の横にIPが表示されます。

Macで自分のIPアドレスを見つける方法

Wi-Fiに接続している場合、MacでIPアドレスを見つける方法：

  1. Appleメニューで [システム環境設定] を選択します。
  2. [ネットワーク] アイコンをダブルクリックします。
  3. ウィンドウの左側にある [Wi-Fi] を選択します—ウィンドウの中央にIPアドレスが表示されます。

イーサネットに接続している場合、MacでIPアドレスを見つける方法：
  1. Appleメニューで [システム環境設定] を選択します。

  3. ウィンドウの左側にある [イーサネット] を選択します—ウィンドウの中央にIPアドレスが表示されます。

AndroidフォンでIPアドレスを見つける方法：

  1. お使いの携帯電話の設定に移動します。
  2. [デバイスについて] を選択します。
  3. [ステータス] をタップします。
  4. ここでは、IPアドレスなどのデバイスに関する情報を見つけることができます。

iPhoneでIPアドレスを見つける方法：
  1. [設定] に移動します。
  2. [Wi-Fi] を選択します。
  3. 接続しているネットワークをタップします。
  4. [IPv4アドレス] の下にIPが表示されます。

IPv4アドレスとIPv6アドレスの違い

IPアドレスの見た目は、使用するインターネットプロトコルのバージョン（IPv4またはIPv6）によって異なります。

IPv4

ほとんどのインターネットサービスプロバイダーは、現在もIPv4を使用しています。IPv4は32バイナリービットに基づいており、例えば103.86.98.1のようにドットで区切られた0から255までの4つの数字で構成されています。



ただし、IPv4で生成できる一意のIPアドレスは43億種類しかありません。IPv4アドレスは昨今のデジタル時代にはすぐに枯渇してしまいます。

IPv6

IPv6アドレスは、8つのブロックに分かれた数字で構成されています。各ブロックは、例えば2400:BB40:1100:0000:0000:0000:0000:
0001のように4桁の16進数で記述され、コロン（:）で区切られています。



多くの場合、0のみの数字グループは、領域を節約するために省略されます。その代わりに、例えば2400:BB40:1100::1のようにギャップを示すコロンが追加されます。

IPv6がまだ完全に実装されていない理由

IPv4プロトコルとは異なり、IPv6では約3.4×10^38種類のIPアドレスを使用できるため、一意のIPアドレスが枯渇することはありません。また、IPv6はより効率的な技術なので、品質と接続性が向上すると主張する人もいます。

しかし、IPv6は次の2つの理由のため、まだ完全には実装されていません。

1. IPv6にはIPv4との後方互換性がありません。お使いのデバイスがIPv6プロトコルで動作している場合、IPv4を使用するWebサイトにはアクセスできません。

2. すぐに利益の出ない新技術を導入することは困難です。人々はまだIPv4で事足りているため、限界に達するまで世界的な変化は起きないでしょう。

グローバルIPアドレスとプライベートIPアドレス

「私のIPアドレスとは何ですか」と尋ねられた場合、通常はパブリックIPを意味します。パブリックIPはグローバルIPとも呼ばれ、インターネットサービスプロバイダーから提供されたIPアドレスです。同一ネットワーク上のデバイスは、インターネットにアクセスするときにこのIPアドレスを共有します。

ホームネットワークを例にとってみましょう。ホームネットワークは、パソコンや、電話、タブレット、そしてルーター上のその他すべてのデバイスから構成されています。これらのデバイスがインターネットに接続する際は、ルーターに割り当てられたパブリックIPアドレスが使用されます。

ルーターがネットワーク上の各デバイスにプライベートIP（ローカルIP）を割り当てることで、デバイスはお互いを識別、検出できるようになります。デバイスがインターネットに接続されると、プライベートIPアドレスは非表示のままとなり、ルーターのパブリックIPアドレスのみが表示されます。パブリックIPアドレスには、ユーザーに関する多くの情報が含まれます。簡単なIPアドレス検索を実行するだけで所在地やインターネットプロバイダーが判明し、個人の特定も不可能ではありません。

よくある質問

IPアドレスには所在地やプロバイダー以外にどのような情報が含まれていますか？

あなたのIPアドレスは国、地域、郵便番号、都市など、インターネットを利用する所在地に関する多くの情報を含んでいます。狡猾なウェブサイトでは、こうした情報をさらにCookieやメタデータから取得した他の詳細と比較できるため、ユーザーの身元をより明確に把握することができます。

誰かに私のIPアドレスを利用される危険はありますか？

第三者が実際にあなたのIPアドレスを乗っ取り、自分のものとして使用する可能性はほとんどありません。ただし一部のユーザーはあなたのIPアドレスにアクセスし、関連するデータを収集することができます。たとえば、ISP（インターネットサービスプロバイダ）はウェブ上でユーザーがアクセスしたウェブサイトをいつでも確認することができます。同時に、訪問するウェブサイトはユーザーの所在地や行動を追跡し、提供するコンテンツの内容をカスタマイズすることができます。たとえば航空券の検索中に高所得のヨーロッパの国からインターネットを利用していることが検出された場合、ウェブサイトは自動的に料金を値上げすることがあります。

IPアドレスを保護するにはどうすればよいですか？

IPアドレスを非表示にすることで保護することができます。VPNサービスを介してこれを行います。このサービスでは接続を別のIPアドレスを介してルーティングすることで、実際のIPアドレスを非表示にします。それ以外にも、VPNには数多くの利点があります。インターネットトラフィックを暗号化して第三者の読み取りやハッカーの悪用を防ぎ、オンラインのプライバシーとセキュリティを向上させます。

IPアドレスはどのように割り当てられるのですか？

世界中のIPアドレスは、IANA（インターネット番号割当機関）の機能を実行している非営利法人ICANNが管理しています。その下には、「地域インターネットレジストリ（RIR）」という世界の各大陸ごとにIPアドレスを割り当てる5つの組織が存在します。この地域インターネットレジストリにより、より小さなIPアドレスブロックがISPやその他のネットワーク事業者に割り当てられます。
